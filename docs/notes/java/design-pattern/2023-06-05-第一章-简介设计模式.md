# 第一章. 简介设计模式

## 1. 什么是设计模式

* 设计模式是系统中针对场景做的解决方案，可以解决功能逻辑开发中的共性问题
* 它仅仅是一种开发指导思想，一种解决对应问题的概念模型，没有绝对的固定格式

## 2. 设计模式分类

* 创建型模式
	1. 单例
	2. 简单工厂
	3. 工厂方法
	4. 抽象工厂
	5. 生成器
	6. 原型模式
* 结构模式
	1. 适配器
	2. 桥接
	3. 组合
	4. 装饰
	5. 外观
	6. 享元
	7. 代理
* 行为模式
	1. 责任链
	2. 命令
	3. 解释器
	4. 迭代器
	5. 中介者
	6. 备忘录
	7. 观察者
	8. 状态
	9. 策略
	10. 模板方法
	11. 访问者
	12. 空对象
